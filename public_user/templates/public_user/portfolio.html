simplejQueryTranslator

simplejQueryTranslator


During development of wake-up-happy project We wanted the site to be translated into several languages. We didn't find any package that would fully satisfy our needs, so We developed simplejQueryTranslator.

It translates text values of elements, as well as titles, placeholders and alt attributes. You can include variables that won't get translated ( user name or city ...), as well as variables that needs translating ( single bedroom or double bedroom ...)

Live demo | GitHub
jQuery, HTML

wake-up-happy

Second milestone project at Code Institute.


We were inspired to create this application to help thousands of families to create additional income for themselves, by renting out a room in their home.
It is a booking site, where customers can find rooms with the view...
and room owners can :

1. add their rooms onto the site, using our 5 step method, with visual and written guides for every step of the journey.

2. log in anytime to see total income and occupancy for the room, and any bookings with customer details.

3. edit room details or add another room

To test it as customer:

1. Search for the room in your desired location in the seach form or on the map
2. Make a purchase with dummy data.
3. You should see succes confirmation pop-up or alert if some of the fields are missing

To test it as an owner of the room:

1. After you clicked on word OWNER? in the top nav follow the guide to add your property to our site.
You can always click on how-to to see demo helping you completing the form.
2. On the payment page, fill in form with dummy data, but remember email and password, so that you will be able to log-in and logout of your property.
3, You can add multiple properties.
4. You can edit or delete your properties.
5. If you book your property, you will see the booking in your dashboard, with customer details: name, email, weeks booked, board type, any requests and date of the booking. We will also show you your occupancy %, and total income.

Visit website | GitHub
Languages used HTML5, CSS3, javascript-jQuery
